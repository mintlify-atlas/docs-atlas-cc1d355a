## Overview

Workspaces are containers that organize agents, tasks, notes, and codebases. Each workspace represents a project or collection of related work.

## Workspace Status

| Status | Description |
|--------|-------------|
| `active` | Workspace is currently active |
| `archived` | Workspace has been archived |

## List Workspaces

<CodeGroup>
```bash cURL
curl http://localhost:3000/api/workspaces
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/api/workspaces');
const data = await response.json();
console.log(data.workspaces);
```

```python Python
import requests

response = requests.get('http://localhost:3000/api/workspaces')
workspaces = response.json()['workspaces']
```
</CodeGroup>

### Endpoint

<ParamField path="GET /api/workspaces" type="GET">
  List all workspaces
</ParamField>

### Query Parameters

<ParamField query="status" type="string">
  Filter by status: `active` or `archived`
</ParamField>

### Response

<ResponseField name="workspaces" type="array" required>
  Array of workspace objects
</ResponseField>

### Workspace Object

<ResponseField name="id" type="string" required>
  Unique workspace identifier
</ResponseField>

<ResponseField name="title" type="string" required>
  Workspace title
</ResponseField>

<ResponseField name="repoPath" type="string">
  Path to associated repository
</ResponseField>

<ResponseField name="branch" type="string">
  Git branch name
</ResponseField>

<ResponseField name="status" type="string" required>
  Workspace status: `active` or `archived`
</ResponseField>

<ResponseField name="metadata" type="object" required>
  Additional metadata as key-value pairs
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of creation
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of last update
</ResponseField>

## Create Workspace

<CodeGroup>
```bash cURL
curl -X POST http://localhost:3000/api/workspaces \
  -H "Content-Type: application/json" \
  -d '{
    "title": "My Project",
    "repoPath": "/path/to/repo",
    "branch": "main",
    "metadata": {"description": "Main workspace"}
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/api/workspaces', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    title: 'My Project',
    repoPath: '/path/to/repo',
    branch: 'main',
    metadata: { description: 'Main workspace' }
  })
});
const data = await response.json();
console.log(data.workspace.id);
```

```python Python
import requests

response = requests.post(
    'http://localhost:3000/api/workspaces',
    json={
        'title': 'My Project',
        'repoPath': '/path/to/repo',
        'branch': 'main',
        'metadata': {'description': 'Main workspace'}
    }
)
workspace = response.json()['workspace']
print(workspace['id'])
```
</CodeGroup>

### Endpoint

<ParamField path="POST /api/workspaces" type="POST">
  Create a new workspace
</ParamField>

### Request Body

<ParamField body="title" type="string" required>

<ParamField body="repoPath" type="string">
  Path to associated repository
</ParamField>

<ParamField body="branch" type="string">
  Git branch name
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata as key-value pairs
</ParamField>

### Response

### Response Example

  Get a single workspace with its codebases
</ParamField>

### Path Parameters

  The workspace ID
</ParamField>

### Response

<ResponseField name="workspace" type="object" required>
  The workspace object
</ResponseField>

<ResponseField name="codebases" type="array" required>
  Array of codebase objects associated with this workspace
</ResponseField>

```json

    "id": "workspace-123",

    "status": "active",
    "createdAt": "2026-03-03T10:00:00.000Z",
```

### Status Codes

| Status Code | Description |
|-------------|-------------|
| `200` | Success |
| `404` | Workspace not found |

## Archive Workspace

<CodeGroup>
```bash cURL
curl -X POST http://localhost:3000/api/workspaces/workspace-123/archive \
  -H "Content-Type: application/json" \
  -d '{"archive": true}'
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/api/workspaces/workspace-123/archive', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ archive: true })
});
const data = await response.json();
console.log(data.workspace.status); // "archived"
```

```python Python
import requests

response = requests.post(
    'http://localhost:3000/api/workspaces/workspace-123/archive',
    json={'archive': True}
)
workspace = response.json()['workspace']
print(workspace['status'])  # "archived"
```
</CodeGroup>

### Endpoint

<ParamField path="POST /api/workspaces/{id}/archive" type="POST">
  Archive or unarchive a workspace
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  The workspace ID
</ParamField>

### Request Body

<ParamField body="archive" type="boolean">
  Set to `true` to archive, `false` to unarchive
</ParamField>

### Response

<ResponseField name="workspace" type="object" required>
  The updated workspace with new status
</ResponseField>

### Response Example

```json
{
  "workspace": {
    "id": "workspace-123",
    "title": "My Project",
    "status": "archived",
    "createdAt": "2026-03-03T10:00:00.000Z",
    "updatedAt": "2026-03-03T12:00:00.000Z"
  }
}
```
